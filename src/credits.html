Credits

RoleDeck stands on the shoulders of excellent open-source work.

Based on AWS Extend Switch Roles

RoleDeck is an unofficial, independent fork of AWS Extend Switch Roles by Toshimitsu Takahashi (Tilfin Ltd.), released under the MIT License. The original project provides the core role-switching engine that RoleDeck builds on. Huge thanks to its author and contributors.

RoleDeck is not affiliated with, sponsored by, or endorsed by Tilfin Ltd. or Amazon Web Services. If you find RoleDeck useful, please consider supporting the original project.
