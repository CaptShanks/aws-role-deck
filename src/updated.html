RoleDeck

A faster, safer way to switch AWS IAM roles.

What RoleDeck adds

  • Color-coded Firefox containers — open each profile in its own isolated container tab; child tabs stay in the same container.
  • Production guardrails — confirm before switching into prod, plus a persistent in-console PRODUCTION banner and an optional idle auto-lock.
  • Favorites & recents — star the roles you use most; recent ones float to the top.
  • Command-palette popup — fuzzy search with full keyboard control, including Alt+1–9 quick-switch.
  • Session auto-renew — silently re-assume the role before the 1-hour limit so you stay signed in.
  • Config upload & discovery — drag-and-drop your ~/.aws/config to auto-discover switchable roles, plus a visual profile editor and a config linter.
  • Deep links, activity log & managed config — land on a specific console page, keep a private switch history, or fetch a team-managed config from a URL.

Built on AWS Extend Switch Roles

RoleDeck is an unofficial fork of AWS Extend Switch Roles by Tilfin Ltd. (MIT). See the Credits page from the popup for full attribution.

All settings are available to everyone — no supporter gate.
